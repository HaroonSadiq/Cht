Architecture

Event-driven by design. Scales from day one.

Every inbound event flows through a hardened five-stage pipeline: ingress, queue, engine, state, egress. Sub-second end-to-end, millions of events per day, never a dropped message.

Vercel Edge

Verifies x-hub-signature-256, returns 200 in <2s.

⇲

Queue

Upstash Redis

// Meta webhook — verify, enqueue, ack in <2s
export async function POST(req: Request) {
  const raw = await req.text();
  const sig = req.headers.get('x-hub-signature-256');

  if (!verifyMetaSignature(raw, sig)) {
    return new Response('Unauthorized', { status: 401 });
  }

  const payload = JSON.parse(raw);

  // Push to Upstash queue for async processing
  await redis.lpush('msg_queue', JSON.stringify({
    event: payload,
    receivedAt: Date.now(),
    idempotencyKey: payload.entry[0].id,
  }));

  return new Response('EVENT_RECEIVED', { status: 200 });
}
lib/engine/traverseFlow.ts
TypeScript
// Flow engine — stateless DAG traverser
export async function traverseFlow({
  state, message, flow,
}: TraverseArgs): Promise<StepResult> {

  const node = flow.nodes[state.currentNodeId];

  switch (node.type) {
    case 'send_message':
      await sender.send(state.contactId, node.content);
      return { next: node.nextStepId };

    case 'branch':
      return { next: evaluateBranches(node.branches, message) };

    case 'delay':
      return { waitUntil: Date.now() + node.durationMs };

    case 'ai_agent':
      const reply = await llm.complete(node.prompt, state.context);
      await sender.send(state.contactId, { text: reply });
      return { next: node.nextStepId };
  }
}

Is FlowBot compliant with Meta's policies?
Yes. We're a Meta Business Partner and built on the official Graph API. We respect the 24-hour messaging window, implement message tags for legitimate out-of-window sends, and never scrape or spoof accounts.
How long does setup take?
Most customers send their first automated flow within 30 minutes of signup. Connecting an Instagram Business or WhatsApp account is one click; templates give you a working revenue flow instantly.
Can I migrate from ManyChat or Chatfuel?
One-click import. We translate flows, contact tags and custom fields automatically, and run them in parallel until you're ready to cut over.
What about TikTok?
TikTok's messaging API is limited — we focus on comment triggers and TikTok Shop customer service, where real revenue lives. We're transparent about what's possible.
Is my customer data safe?
SOC 2 Type II certified, GDPR and CCPA compliant. Tokens encrypted at rest with envelope encryption. Your data is never used to train shared AI models.
